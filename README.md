# BlankSoftware Raid Connect
Modification of the connect interface for the main DayZ Standalone raid settings mod [Blanksoftware_RaidSettings]. ## Installation [For Clients]
You must have the [Core](https://discord.gg/xt2GGzfFY7) installed.

- You have `Blanksoftware_RaidSettings.pbo` and the `BlankSoftware_RaidConnect` folder. All modifications to the mod are made by editing `BlankSoftware_RaidConnect`.
- To customize the mod for yourself, open `BlankSoftware_RaidConnect` and edit what you need, then pack it using your tools (e.g., Addon Builder, PBO Project, PBO Tools, Mikero Tools [ATTENTION! Do not pack via PBO Manager!!]).
- Sign `BlankSoftware_RaidConnect.pbo` with your own key using DS Utils (DayZ Tools).
- Place both PBOs and their associated keys into your client mod.
- Done.

## Attention
- When packing `BlankSoftware_RaidConnect`, pay attention to the parameters of the saved files, specify the following:
```*.csv;*.layout;*.fnt;*.xml;*.tga;*.rvmat;*.emat;*.edds;*.paa;*.ptc;*.c;*.imageset;*.layout;*.ogg;*.json;*.mp4```
- Editing of almost any text is done by editing the `stringtable.csv` translation file at the root of `BlankSoftware_RaidConnect`.
I advise you to use the String Table Editor program for editing, link to the guide: [DISCORD](https://discord.com/channels/1201511657990520872/1224962901975240774)
